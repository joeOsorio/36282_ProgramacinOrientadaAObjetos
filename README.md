# Programaci-n-Orientada-a-Objetos
Repositorio académico de Programación Orientada a Objetos. Contiene prácticas, ejercicios y proyectos desarrollados en Java SE (JDK 8+), aplicando los principios de la POO. El código sigue buenas prácticas y criterios de legibilidad, tomando como referencia los estándares de codificación ANSI C para mantener orden, claridad y estructura.
