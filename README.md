📘 Programación Orientada a Objetos – Repositorio Académico

Este repositorio contiene el desarrollo de prácticas, ejercicios, ejemplos y proyectos correspondientes a la materia Programación Orientada a Objetos (POO), cursada a nivel universitario.
El objetivo principal es aplicar los fundamentos del paradigma orientado a objetos mediante el uso de un lenguaje de programación estructurado y orientado a objetos, promoviendo buenas prácticas de diseño, claridad en el código y pensamiento lógico.

🎯 Objetivos de la materia

Comprender y aplicar los principios de la Programación Orientada a Objetos: encapsulamiento, abstracción, herencia y polimorfismo.

Diseñar soluciones utilizando clases, objetos y relaciones entre ellos.

Desarrollar programas claros, estructurados y mantenibles.

Fomentar el análisis, la reutilización de código y el uso de estándares de programación.

🛠️ Tecnologías utilizadas

Lenguaje principal:

Java SE (Java Standard Edition)

Compatible con JDK 8 o superior

Paradigma: Programación Orientada a Objetos

📐 Estándares y buenas prácticas

El código en Java sigue un estilo simple, legible y estructurado, adecuado para el aprendizaje académico.

Se aplican principios de buen diseño orientado a objetos y documentación básica del código.

Como referencia de estilo y disciplina en la escritura de código, se consideran los estándares de codificación ANSI C, especialmente en:

Uso adecuado de indentación

Nombres descriptivos para variables y métodos

Claridad en la estructura lógica del programa

Comentarios explicativos cuando son necesarios

(Aunque ANSI C corresponde a otro lenguaje, sus principios de orden, claridad y estructura se toman como base conceptual para la programación.)
